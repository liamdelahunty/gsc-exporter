Weekly Editorial Performance Guide

A guide for accountancydaily.co to identify which news stories and articles performed best across Web, Discover, and Google News.

Core Editorial Reports

Since editorial focus is on story performance, these reports provide the most granular view of which pages "won" during the week. Note: You no longer need to manually calculate dates; use the --last-7-days flag to automatically anchor to the latest available data.

1. Page-Level Performance (The "What Worked" List)

Run this for each search type. The --last-7-days flag will automatically find the most recent 7-day window with complete data.

Web Search (Standard SEO)
python reports/page_level_report.py sc-domain:accountancydaily.co --search-type web --last-7-days
Google Discover (Viral/Interest Traffic)
python reports/page_level_report.py sc-domain:accountancydaily.co --search-type discover --last-7-days
Google News (Trending Traffic)
python reports/page_level_report.py sc-domain:accountancydaily.co --search-type news --last-7-days
2. Snapshot Report (High-Level Summary)

Best for a quick executive summary of the week's totals (Total Clicks, Top 10 Pages, Top 10 Queries).

python reports/snapshot_report.py sc-domain:accountancydaily.co --last-7-days
3. Queries-Pages Analysis (Keyword Context)

Useful for seeing exactly which search terms led readers to specific news stories.

python reports/queries_pages_analysis.py sc-domain:accountancydaily.co --last-7-days

Smart Date Detection

The GSC Exporter now automatically handles the 48 to 72-hour data lag. When you run a report without explicit dates or with the --last-7-days flag, it follows this logic:

  • Latest Data Check: It automatically queries GSC to find the most recent date with available data.
  • Automatic Range:
    • Using --last-7-days: Returns the 7 days ending at the latest available date.
    • No flags / Default: Returns the last complete available calendar month.

Example: If today is Monday, June 1st and data is only available up to May 29th, the --last-7-days flag will automatically fetch May 23rd to May 29th.

Strategic Insight: News vs. Momentum

For a news site like accountancydaily.co, "momentum" (comparing this week to last) is often less relevant than volume and reach of individual stories. Viral news spikes in Discover or News often don't repeat, so focus on:

  • Discover Clicks: Indicates high interest/headline clickability.
  • News Impressions: Indicates how well the site is appearing in "Top Stories" carousels.
  • Query Count (Web): High query counts on a single page suggest a broad, evergreen topic.
