How to Read the Monthly Summary Report

This report provides a concise overview of key Google Search Console performance metrics for the last complete calendar month across all (or a specified) property.

Key Metrics Explained:

  • Property: The Google Search Console property URL (e.g., https://www.example.com/ or sc-domain:example.com).
  • Clicks: The total number of clicks from Google Search results to your site.
  • Impressions: The total number of times your site appeared in Google Search results.
  • CTR (Click-Through Rate): The percentage of impressions that resulted in a click (Clicks / Impressions).
  • Position: The average ranking position of your site in Google Search results. Lower numbers are better.
  • # queries: The number of unique search queries that led to impressions for your site.
  • # pages: The number of unique pages on your site that received traffic from Google Search.

Truncated Data Warning (*)

If you see an asterisk (*) next to a "# queries" or "# pages" value, it indicates that the count might be incomplete. This occurs when the number of unique queries or pages for that property exceeds the export limit of the Google Search Console API. While the script attempts to paginate for a more accurate count, very large datasets can still hit underlying API limitations, meaning the true number might be higher than reported.
